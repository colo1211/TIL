# 1. TypeScript 연습

### TypeScript 환경설정

1. 
```
$ mkdir ts-practice # ts-practice 라는 디렉터리 생성
$ cd ts-practice # 해당 디렉터리로 이동
$ npm init -y # pakage.json 파일을 만들기 위해
```

2. 
```
npm install -g typescript
```

3. `tsc --init` 명령을 통해서 tsconfig.json 파일을 생성
```
{
  "compilerOptions": {
    "target": "es5",
    "module": "commonjs",
    "strict": true,
    "esModuleInterop": true
  }
}
```
